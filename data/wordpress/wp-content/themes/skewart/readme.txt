A dark, cyberpunk style WordPress theme with glitch effects and warm gradients.

== Description ==
Skewart is a dark, cyberpunk-inspired WordPress theme featuring futuristic glitch effects, neon accents, and dynamic gradients. Designed with a modern digital aesthetic, it creates an immersive experience perfect for tech blogs, gaming sites, creative portfolios, digital art showcases, and forward-thinking businesses.

The theme's distinctive visual style combines a dark background palette with vibrant cyan and magenta highlights, creating the atmosphere of a high-tech digital world. Carefully crafted typography using Orbitron and Oswald fonts enhances the futuristic feel while maintaining excellent readability.

Key Features:
* Striking cyberpunk aesthetic with glitch effects and neon accents
* Fully responsive design that looks great on all devices
* Custom header with support for background images
* Distinctive typography using Google Fonts (Orbitron and Oswald)
* Support for a custom logo and site icon
* Dedicated widget area in the footer
* Custom navigation menu with dropdown support
* Featured image support with cyberpunk styling
* Dark color scheme optimized for reduced eye strain
* Block editor styles for a consistent editing experience
* Translation-ready for international audiences
* SEO-friendly semantic markup
* Customizable background options
* Fast loading times and optimized performance
* Compatible with popular WordPress plugins
* Thoroughly tested with the latest WordPress version

Whether you're creating a tech blog, gaming site, digital art portfolio, or any project that calls for a bold, futuristic design, Skewart provides the perfect backdrop for your content with its immersive cyberpunk atmosphere.

== Installation ==
1. Upload the theme files to the `/wp-content/themes/skewart` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Themes' screen in WordPress.
3. Configure theme options through the WordPress Customizer.

== Frequently Asked Questions ==

= How do I set up the menu? =
You can set up the menu by navigating to Appearance > Menus in your WordPress dashboard.

= How do I change the logo? =
You can upload a custom logo by going to Appearance > Customize > Site Identity.
